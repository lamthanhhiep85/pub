I — Lý thuyết đường cong trong ℝ³
Đường cong (Curve) — Định nghĩa & Tham số hóa Curves
Đường cong chính quy:
γ: I → ℝⁿ, γ'(t) ≠ 0 ∀t ∈ I

Độ dài cung (Arc length):
s(t) = ∫_a^t |γ'(u)| du

Tham số hóa theo độ dài cung:
γ̃(s) = γ(t(s)), |γ̃'(s)| = 1

Vận tốc & Tốc độ:
v(t) = γ'(t),   speed = |γ'(t)|

Tam diện Frenet–Serret (Frenet frame)

T = γ'/|γ'|  // unit tangent (tiếp vectơ)
N = T'/|T'|  // principal normal (pháp vectơ chính)
B = T × N  // binormal (phụ pháp vectơ)

{T, N, B} trực chuẩn, thuận chiều
Độ cong & Xoắn (Curvature & Torsion) Frenet–Serret
Độ cong (Curvature) κ:
κ = |T'(s)| = |γ'' (s)|  // theo param s
κ = |γ' × γ''| / |γ'|³  // param bất kỳ

Xoắn (Torsion) τ:
τ = −N'·B / 1 = (γ'×γ'')·γ''' / |γ'×γ''|²

Công thức Frenet–Serret:
T' = κN
N' = −κT + τB
B' = −τN

Định lý cơ bản về đường cong

Cho κ(s) > 0, τ(s) liên tục
⟹ ∃! đường cong (sai khác chuyển động cứng)
có κ và τ đó

Đặc biệt:
κ = const, τ = 0: đường tròn
κ = const, τ = const ≠ 0: đường xoắn ốc (helix)
κ = 0: đường thẳng
Mặt phẳng osculating Planes
Osculating plane (mặt phẳng hôn):
span{T, N} — mặt phẳng cong nhất

Normal plane:
span{N, B} ⊥ T

Rectifying plane:
span{T, B}

Radius of curvature (Bán kính cong): ρ = 1/κ

Đường cong phẳng Plane Curves
γ(t) = (x(t), y(t))

κ = (x'y'' − y'x'') / (x'²+y'²)^{3/2}

Winding number:
n(γ, p) = (1/2π)∮ dθ

Rotation index:
i(γ) = (1/2π)∮ κ ds ∈ ℤ

II — Lý thuyết mặt trong ℝ³
Mặt chính quy & First Fundamental Form Surfaces I
Tham số hóa mặt:
σ: U ⊂ ℝ² → ℝ³, chính quy: σ_u × σ_v ≠ 0

First fundamental form I (Dạng cơ bản I):
I = E du² + 2F du dv + G dv²

E = σ_u·σ_u,  F = σ_u·σ_v,  G = σ_v·σ_v

Ma trận metric:
g_{ij} = [E F]
         [F G]

Diện tích:
A = ∬ |σ_u × σ_v| du dv = ∬ √(EG−F²) du dv

Độ dài đường cong trên mặt:
L = ∫ √(Eů²+2Fův̇+Gv̇²) dt
Second Fundamental Form & Độ cong mặt Surfaces II
Vectơ pháp tuyến đơn vị:
n = (σ_u × σ_v) / |σ_u × σ_v|

Second fundamental form II (Dạng cơ bản II):
II = L du² + 2M du dv + N dv²

L = σ_{uu}·n,  M = σ_{uv}·n,  N = σ_{vv}·n

Toán tử hình dạng (Shape operator / Weingarten map):
S = −dn: TₚM → TₚM
Sv = −(dn)(v)

Eigenvalue của S = Độ cong chính (Principal curvature):
κ₁, κ₂ — hướng chính e₁, e₂
Độ cong Gauss & Mean Curvatures
Gaussian curvature (Độ cong Gauss):
K = κ₁κ₂ = det(II)/det(I)
K = (LN−M²)/(EG−F²)

Mean curvature (Độ cong trung bình):
H = (κ₁+κ₂)/2
H = (EN−2FM+GL) / 2(EG−F²)

Umbilic point (Điểm rốn):
κ₁ = κ₂ ← tất cả hướng là chính

K > 0: elliptic · K = 0: parabolic · K < 0: hyperbolic

Phương trình Euler & Lines of curvature Curvature lines
Độ cong pháp tuyến (Normal curvature):
κ_n(v) = II(v,v) / I(v,v)

Định lý Euler:
κ_n(θ) = κ₁cos²θ + κ₂sin²θ

Lines of curvature:
Đường mà tiếp tuyến là hướng chính

Asymptotic lines:
κ_n = 0 → II(v,v) = 0

Đường trắc địa (Geodesics) Geodesics
Định nghĩa: Đường cong γ trên mặt thỏa
∇_{γ'} γ' = 0 (song song vận chuyển tiếp tuyến)

Geodesic curvature (Độ cong trắc địa):
κ_g = γ'' · (n × γ')
γ là geodesic ⟺ κ_g = 0

Phương trình geodesic:
ü^k + Γ^k_{ij} u̇^i u̇^j = 0
// Γ^k_{ij} = Christoffel symbols

Geodesic trên mặt cầu: đại vòng (great circle)
Geodesic trên mặt phẳng: đường thẳng
Geodesic trên cylinder: đường xoắn ốc
Định lý Gauss–Bonnet Global Theorem
Gauss–Bonnet (cục bộ):
∬_R K dA + ∮_{∂R} κ_g ds + Σ θᵢ = 2π
// θᵢ = góc ngoài tại đỉnh

Gauss–Bonnet (toàn cục):
∬_M K dA = 2π χ(M)
// χ = Euler characteristic

Hệ quả:
∬_{S²} K dA = 4π  (χ = 2)
∬_{Torus} K dA = 0  (χ = 0)
Tam giác cầu: tổng góc = π + ∬_R K dA > π

Đây là cầu nối lớn nhất giữa hình học vi phân cục bộ và topo toàn cục.

III — Đa tạp mượt (Smooth Manifolds)
Định nghĩa đa tạp & Atlas Manifolds
Topological manifold M^n:
Hausdorff, 2nd countable, locally ≅ ℝⁿ

Chart (Bảng đồ):
(U, φ), U ⊂ M mở, φ: U → ℝⁿ homeomorphism

Atlas (Atlat):
𝒜 = {(Uα, φα)} che phủ M

Smooth manifold (Đa tạp mượt):
Transition maps (ánh xạ chuyển tiếp):
φ_β ∘ φ_α⁻¹: φ_α(U_α∩U_β) → φ_β(U_α∩U_β)
là C∞ diffeomorphism

Đa tạp con (Submanifold):
N ⊂ M submanifold nếu inclusion mượt

Ví dụ đa tạp kinh điển

ℝⁿ Sⁿ (n-sphere) Tⁿ (n-torus) ℝPⁿ ℂPⁿ GL(n,ℝ) SO(n) SU(n) Grassmannian
Không gian tiếp tuyến & Vectơ tiếp tuyến Tangent Space
Tangent vector (Vectơ tiếp tuyến) tại p:
v = [γ] = lớp tương đương của đường cong qua p
Hoặc: derivation v: C∞(M) → ℝ
v(fg) = v(f)g(p) + f(p)v(g)

Tangent space (Không gian tiếp tuyến):
TₚM = span{∂/∂x¹|ₚ, …, ∂/∂xⁿ|ₚ}
dim TₚM = dim M

Tangent bundle (Bó tiếp tuyến):
TM = ⊔_{p∈M} TₚM → M

Cotangent space (Không gian đối tiếp):
T*ₚM = (TₚM)*
Cơ sở đối ngẫu: dx^i(∂/∂x^j) = δ^i_j
Ánh xạ mượt & Pullback/Pushforward Maps
Smooth map: F: M → N

Pushforward (Đẩy tiến):
F*: TₚM → T_{F(p)}N
(F*v)(f) = v(f∘F)

Pullback (Kéo lùi):
F*: T*_{F(p)}N → T*ₚM
(F*ω)(v) = ω(F*v)

Immersion: F* injective
Submersion: F* surjective
Diffeomorphism: F smooth bij, F⁻¹ smooth
Tensor & Tensor Field Tensors
Tensor (p,q) tại p:
T: (T*ₚM)^p × (TₚM)^q → ℝ

Ký hiệu: T^{i₁…iₚ}_{j₁…jq}

Metric tensor g (loại (0,2)):
g_{ij} = g(∂ᵢ, ∂ⱼ) — symmetric, positive def.

Raising/lowering indices:
vⁱ = gⁱʲ vⱼ,   vᵢ = gᵢⱼ vʲ

Tensor product: S⊗T, type (p+r, q+s)
Differential Forms (Dạng vi phân) Forms
k-form: ω ∈ Ω^k(M)
ω: TM^k → ℝ, hoàn toàn phản xứng

Exterior derivative (Đạo hàm ngoài):
d: Ω^k → Ω^{k+1},   d² = 0

Wedge product:
α∧β ∈ Ω^{p+q}, phản giao hoán
α∧β = (−1)^{pq} β∧α

de Rham cohomology:
H^k_{dR}(M) = ker d^k / im d^{k-1}

Integration: ∫_M ω, M compact oriented
Vector field & Lie bracket Vector Fields
Vector field (Trường vectơ):
X: M → TM, X(p) ∈ TₚM, section của TM

Integral curve (Đường tích phân):
γ: I → M, γ'(t) = X(γ(t))

Flow (Luồng):
φ_t: M → M, d/dt φ_t(p) = X(φ_t(p))

Lie bracket (Ngoặc Lie):
[X,Y](f) = X(Y(f)) − Y(X(f))

Frobenius theorem:
Distribution Δ involutive
⟺ [X,Y] ∈ Δ ∀X,Y ∈ Δ
⟺ Δ integrable (có integral manifold)
Lie derivative & Cartan's formula Lie Derivative
Lie derivative theo X:
ℒ_X T = d/dt|_{t=0} (φ_t)* T

Trên function:
ℒ_X f = X(f) = df(X)

Trên vector field:
ℒ_X Y = [X, Y]

Cartan's magic formula:
ℒ_X ω = d(ι_X ω) + ι_X(dω)
// ι_X = interior product (tích co)

Hệ quả quan trọng:
ℒ_X(dω) = d(ℒ_X ω)
IV — Hình học Riemann (Riemannian Geometry)
Metric Riemann & Kết nối Levi-Civita Riemannian
Riemannian metric (Metric Riemann):
g: TM × TM → ℝ
symmetric, bilinear, positive definite

Christoffel symbols (Ký hiệu Christoffel):
Γ^k_{ij} = (1/2) g^{kl} (∂_i g_{jl} + ∂_j g_{il} − ∂_l g_{ij})

Levi-Civita connection ∇:
(1) Torsion-free: ∇_X Y − ∇_Y X = [X,Y]
(2) Metric compatible: ∇g = 0
    X g(Y,Z) = g(∇_X Y, Z) + g(Y, ∇_X Z)

Covariant derivative:
∇_X Y = (X^i ∂_i Y^k + X^i Y^j Γ^k_{ij}) ∂_k
Song song vận chuyển & Geodesic Parallel Transport
Parallel transport dọc γ:
∇_{γ'} V = 0  // V song song dọc γ

dV^k/dt + Γ^k_{ij} (dx^i/dt) V^j = 0

Geodesic (tổng quát):
∇_{γ'} γ' = 0
d²x^k/dt² + Γ^k_{ij} (dx^i/dt)(dx^j/dt) = 0

Geodesic = "đường thẳng" trên M
Cực tiểu hóa cục bộ độ dài:
δ ∫ √(g_{ij} ẋ^i ẋ^j) dt = 0

Holonomy (Toàn luân):
Hol(M,g,p) ≤ O(n) — nhóm phép quay
từ parallel transport trên các vòng khép kín
V — Độ cong (Curvature Tensors)
Riemann Curvature Tensor & các loại độ cong Curvature
Riemann curvature tensor (Tensor độ cong Riemann):
R(X,Y)Z = ∇_X∇_Y Z − ∇_Y∇_X Z − ∇_{[X,Y]} Z

Thành phần:
R^l_{kij} = ∂_i Γ^l_{jk} − ∂_j Γ^l_{ik} + Γ^l_{im}Γ^m_{jk} − Γ^l_{jm}Γ^m_{ik}

Tính đối xứng:
R_{ijkl} = −R_{jikl} = −R_{ijlk} = R_{klij}

Bianchi identity (1):
R(X,Y)Z + R(Y,Z)X + R(Z,X)Y = 0

Bianchi identity (2) — differential:
(∇_X R)(Y,Z) + (∇_Y R)(Z,X) + (∇_Z R)(X,Y) = 0
Ricci tensor (Tensor Ricci):
Ric_{ij} = R^k_{ikj} = R^k_{\ ikj}
Ric(X,Y) = tr[Z ↦ R(Z,X)Y]

Ricci scalar (Scalar Ricci / Curvature scalar):
R = g^{ij} Ric_{ij} = tr_g Ric

Sectional curvature (Độ cong mặt cắt):
K(σ) = g(R(X,Y)Y, X) / (g(X,X)g(Y,Y)−g(X,Y)²)
σ = span{X,Y} ⊂ TₚM

Einstein tensor (Tensor Einstein):
G_{ij} = Ric_{ij} − (1/2) g_{ij} R
∇^i G_{ij} = 0  // bảo toàn tự động

Einstein field equations:
G_{μν} + Λg_{μν} = 8πG/c⁴ · T_{μν}
Space forms Constant Curvature
K = const trên M → "Space form"

K = +1: Sⁿ (n-sphere)
K =  0: ℝⁿ (Euclidean)
K = −1: ℍⁿ (Hyperbolic space)

K(σ) = g(R(e₁,e₂)e₂, e₁)
khi K = c: R_{ijkl} = c(g_{ik}g_{jl}−g_{il}g_{jk})
Jacobi field & Conjugate points Jacobi
Jacobi equation:
J'' + R(J,γ')γ' = 0
J: biến phân của geodesic

Conjugate points:
p,q liên hợp nếu J(0)=J(l)=0
J ≠ 0 trên (0,l)

Bonnet–Myers:
Ric ≥ (n−1)K > 0
⟹ M compact, diam ≤ π/√K
So sánh độ cong Comparison
Rauch comparison:
Nếu K ≤ K₀ thì Jacobi fields
của M ≥ của M_{K₀}

Cartan–Hadamard:
M hoàn chỉnh, K ≤ 0
⟹ M̃ = ℝⁿ (simply conn.)

Cheeger–Gromoll (Soul thm):
M open, K ≥ 0 ⟹ M deformation
retract lên compact "soul"
VI — Kết nối (Connections) & Fiber Bundles
Affine Connection & Curvature tổng quát Connections
Connection ∇ trên vector bundle E:
∇: Γ(E) → Γ(T*M ⊗ E)
∇(fs) = df⊗s + f∇s

Curvature của connection:
F∇(X,Y)s = ∇_X∇_Y s − ∇_Y∇_X s − ∇_{[X,Y]}s

Gauge transformation:
∇ ↦ g⁻¹∇g = ∇ + g⁻¹dg

Yang–Mills functional:
YM(∇) = ∫_M |F∇|² dvol
Yang–Mills eq.: d∇ *F∇ = 0
Principal Bundle & Connection form Bundles
Principal G-bundle:
P →^π M, G tác dụng freely trên P

Connection 1-form (Ehresmann):
ω ∈ Ω¹(P; 𝔤)
ω(v*) = v ∀v ∈ 𝔤, (R_g)*ω = Ad_{g⁻¹}ω

Curvature form (Cartan):
Ω = dω + (1/2)[ω,ω] ∈ Ω²(P; 𝔤)

Bianchi identity:
dΩ + [ω, Ω] = 0

Holonomy group Hol(p) ≤ G
Characteristic Classes (Lớp đặc trưng) Chern · Pontryagin · Euler
Chern classes (Complex bundles):
c_k(E) ∈ H^{2k}(M;ℤ)
c(E) = 1+c₁+c₂+…

Chern–Weil:
c_k = [(i/2π)^k · σ_k(F∇)]
σ_k = k-th elem. symm. poly in eigenvalues

c₁(E) = [i/2π · tr F∇]
Pontryagin classes (Real bundles):
p_k(E) ∈ H^{4k}(M;ℤ)
p_k(E) = (−1)^k c_{2k}(E⊗ℂ)

Euler class:
e(E) ∈ H^n(M;ℤ), E rank n
e(TM) → Euler char χ(M)

∫_M e(TM) = χ(M)
Atiyah–Singer Index Theorem:
ind(D) = ∫_M ch(σ_D) · Td(TM)

Ví dụ: Gauss–Bonnet–Chern
∫_M Pf(Ω/2π) = χ(M)

Hirzebruch signature:
σ(M) = ∫_M L(p₁, p₂, …)

Riemann–Roch:
χ(E) = ∫_M ch(E)·Td(TM)

Atlas với transition maps holomorphic

J: TM → TM, J² = −Id

h(X,Y) = g(X,Y) + iω(X,Y)

Hình học Symplectic Symplectic
Symplectic manifold (M²ⁿ, ω):
ω ∈ Ω²(M), closed (dω=0), non-degenerate

Darboux theorem:
Locally: ω = Σ dpᵢ ∧ dqᵢ
(không có bất biến cục bộ!)

Hamiltonian vector field:
ι_{X_H} ω = −dH
ẋ = ∂H/∂p, ṗ = −∂H/∂q

Poisson bracket:
{f,g} = ω(X_f, X_g) = ∂f/∂q·∂g/∂p − ∂f/∂p·∂g/∂q

Liouville: ω^n preserved by flow
Spin geometry & Dirac Spin
Spin structure:
Lift của SO(n)-frame bundle
đến Spin(n)-bundle

Spinor bundle S → M

Dirac operator:
D = Σ eᵢ · ∇_{eᵢ}: Γ(S)→Γ(S)

// Lichnerowicz formula

R > 0 ⟹ ker D = 0
(không có harmonic spinors)
Sub-Riemannian geometry Sub-Riemannian
(M, Δ, g): Δ ⊂ TM distribution
g metric trên Δ

Nếu [Δ, Δ] = TM
thì (M,d_CC) connected

d_CC(p,q) = inf{len của Δ-curves}

Ứng dụng: cơ học, robot,
nhãn khoa (vision), MRI
