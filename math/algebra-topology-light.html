Tiên đề Group Axioms
∀a,b,c ∈ G:
(1) (a∗b)∗c = a∗(b∗c)  // associativity (kết hợp)
(2) ∃e: e∗a = a∗e = a  // identity (đơn vị)
(3) ∃a⁻¹: a∗a⁻¹ = e   // inverse (nghịch đảo)

Abelian group thêm: a∗b = b∗a (giao hoán)

Tiên đề Topological Space Axioms
τ ⊆ 𝒫(X) là topology khi:
(1) ∅, X ∈ τ  // hai tập đặc biệt
(2) Uᵢ ∈ τ ⟹ ⋃ᵢ Uᵢ ∈ τ  // hợp tùy ý
(3) U,V ∈ τ ⟹ U∩V ∈ τ  // giao hữu hạn

Metric topology: U ∈ τ ⟺ ∀x∈U ∃ε>0: B(x,ε)⊆U

Cầu Nối — Algebraic Topology (Topo Đại Số)
Fundamental Group π₁ — Nhóm cơ bản Homotopy
π₁(X, x₀) = { vòng γ: [0,1]→X, γ(0)=γ(1)=x₀ } / ≃

// phép nhân: ghép vòng (concatenation)
[α]·[β] = [α * β]

// Seifert–van Kampen: nếu X = A∪B, A∩B liên thông
π₁(X) ≅ π₁(A) *_{π₁(A∩B)} π₁(B)  // amalgamated product

Lie Group & Lie Algebra — Cầu nối đặc biệt Smooth
Lie group G: group + smooth manifold, phép nhân mượt
Lie algebra 𝔤 = T_eG: không gian tiếp tuyến tại e

[X, Y] = Lie bracket (ngoặc Lie)
exp: 𝔤 → G  // exponential map (ánh xạ mũ)
exp(tX)·exp(tY) ≈ exp(t(X+Y) + t²/2·[X,Y] + …)
// Baker–Campbell–Hausdorff

Ad: G → GL(𝔤),  ad: 𝔤 → gl(𝔤)
ad_X(Y) = [X, Y]

Galois Theory (Lý thuyết Galois) — Algebra gặp Topo Fields
Field extension (Mở rộng trường): K/F
[K:F] = dimF K  // degree (bậc)

Galois group (Nhóm Galois):
Gal(K/F) = Aut_F(K) = { σ: K→K | σ|_F = id }

Galois correspondence (Tương ứng Galois):
{H ≤ Gal(K/F)} ←→ {F ⊆ L ⊆ K}
H ↦ K^H (fixed field — trường cố định)
L ↦ Gal(K/L)

Solvability (Giải được):
f(x) giải được bằng căn ⟺ Gal(f) là solvable group (nhóm giải được)

Ví dụ: Gal(ℚ(√2,√3)/ℚ) ≅ ℤ/2×ℤ/2 — nhóm Klein V₄
