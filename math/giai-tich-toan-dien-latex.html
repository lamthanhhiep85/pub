Giới hạn (Limit) Foundation
ε-δ definition:
\(\lim_{x\to a} f(x) = L \Leftrightarrow\)
\(\forall \varepsilon >0, \exists \delta >0: 0<|x−a|<\delta \Rightarrow |f(x)−L|<\varepsilon\)

Limit tại \(\infty\):
\(\lim_{x\to \infty } f(x) = L \Leftrightarrow\)
\(\forall \varepsilon >0, \exists M: x>M \Rightarrow |f(x)−L|<\varepsilon\)
Tính chấtCông thức
Tuyến tính\(\lim(\alpha f+\beta g) = \alpha \cdot \lim f + \beta \cdot \lim g\)
Tích\(\lim(f\cdot g) = (\lim f)\cdot (\lim g)\)
Kẹp (Sandwich)\(g\leq f\leq h, \lim g=\lim h=L \Rightarrow \lim f=L\)
L'Hôpital\(0/0 hay \infty /\infty : \lim f/g = \lim f'/g'\)

\(\lim_{x\to 0} \sin(x)/x = 1\)
\(\lim_{x\to 0} (1−\cos x)/x^2 = 1/2\)
\(\lim_{x\to \infty } (1+1/x)^x = e\)
\(\lim_{x\to 0} (eˣ−1)/x = 1\)
\(\lim_{x\to 0} ln(1+x)/x = 1\)
\(\lim_{x\to 0} xˣ = 1\) // \(0^0\) dạng bất định
Liên tục (Continuity) Foundation
f liên tục tại a \(\Leftrightarrow\)
(1) f(a) xác định
(2) \(\lim_{x\to a}\) f(x) tồn tại
\((3) \lim_{x\to a} f(x) = f(a)\)

Infinite
Vô cực		Giới hạn = \(\pm \infty\) (asymptote đứng)

Đạo hàm (Derivative) — Vi phân (Differential) Calculus I
Định nghĩa:
\(f'(x) = \lim_{h\to 0} [f(x+h) − f(x)] / h\)

Vi phân:
df = f'(x) dx

Ký hiệu:
\(f'(x) = df/dx = Df = ẋ (theo t)\)

Quy tắc đạo hàm

Quotient (Thương)\((f/g)' = (f'g−fg')/g^2\)
Chain (Dây chuyền)\((f\circ g)' = f'(g)\cdot g'\)
Inverse (Nghịch)\((f^{-1})'(y) = 1/f'(x)\)

\((eˣ)' = eˣ (aˣ)' = aˣ ln a\)
\((ln x)' = 1/x (\log_a x)' = 1/(x ln a)\)
\((\sin x)' = \cos x (\cos x)' = −\sin x\)

(sec x)' = sec x tan x

\((\arctan x)' = 1/(1+x^2)\)
\((\sinh x)' = \cosh x (\cosh x)' = \sinh x\)
(tanh x)' = \(sech^{2x}\)
Định lý giá trị trung bình MVT
Rolle: \(f(a)=f(b) \Rightarrow \exists c\in (a,b): f'(c)=0\)

MVT (Lagrange):
\(\exists c\in (a,b): f'(c) = [f(b)−f(a)]/(b−a)\)

Cauchy MVT:
\(\exists c: f'(c)/g'(c) = [f(b)−f(a)]/[g(b)−g(a)]\)

Taylor's theorem:
\(f(x) = \Sigma _{k=0}^{n} f^(k)(a)/k! \cdot (x−a)^k + R_n\)
\(R_n = f^(n+1)(c)/(n+1)! \cdot (x−a)^(n+1)\)
\(// Lagrange remainder, c\in (a,x)\)

Giải tích thực (Real Analysis) — nền tảng nghiêm ngặt Real Analysis
Axiom of Completeness:
Mọi tập bị chặn trên \(\subset\) \(\mathbb{R}\) có supremum

Cauchy sequence:
\(\forall \varepsilon >0 \exists N: m,n>N \Rightarrow |a_x−a_m|<\varepsilon\)
\(\mathbb{R} complete: Cauchy \Leftrightarrow convergent\)

Bolzano–Weierstrass:
Mọi dãy bị chặn có dãy con hội tụ
Liên tục đều (Uniform continuity):
\(\forall \varepsilon >0 \exists \delta : |x−y|<\delta \Rightarrow |f(x)−f(y)|<\varepsilon\)

Hội tụ đều (Uniform convergence):
\(f_x \Rightarrow f: \forall \varepsilon \exists N \forall x: |f_x(x)−f(x)|<\varepsilon\)

Weierstrass M-test:
\(|f_x(x)|\leq M_x\), \(\Sigma M_x<\infty\) \(\Rightarrow\) \(\Sigma f_x\) hội tụ đều
Measure theory (Lý thuyết độ đo):
\(\sigma -algebra, measure \mu , L^1(\mu )\)

Lebesgue integral:
Tổng quát hóa Riemann
f đo được, \(\int f\) \(d\mu\)

DCT (Dominated Conv.):
\(f_x\to f p.w., |f_x|\leq g \in L^1\)
\(\Rightarrow \lim\int f_x = \int f\)

Giải tích vectơ — Các định lý lớn Vector Calculus

Toán tử vi phân vectơ

Gradient: \(\nabla f = (\partial f/\partial x, \partial f/\partial y, \partial f/\partial z)\)
Divergence: \(\nabla \cdot F = \partial F_n/\partial x + \partial F_y/\partial y + \partial Fz/\partial z\)
Curl: \(\nabla \times F = \det|i j k / \partial _n \partial _y \partial _z / F_n F_y Fz|\)
Laplacian: \(\nabla ^{2f} = \partial ^{2f}/\partial x^2+\partial ^{2f}/\partial y^2+\partial ^{2f}/\partial z^2\)

Identities:
\(\nabla \times (\nabla f) = 0\) // curl of gradient = 0
\(\nabla \cdot (\nabla \times F) = 0\) // div of curl = 0
