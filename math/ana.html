Giới hạn (Limit) Foundation
ε-δ definition:
lim_{x→a} f(x) = L ⟺
∀ε>0, ∃δ>0: 0<|x−a|<δ ⟹ |f(x)−L|<ε

Limit tại ∞:
lim_{x→∞} f(x) = L ⟺
∀ε>0, ∃M: x>M ⟹ |f(x)−L|<ε
Tính chấtCông thức
Tuyến tínhlim(αf+βg) = α·lim f + β·lim g
Tíchlim(f·g) = (lim f)·(lim g)
Kẹp (Sandwich)g≤f≤h, lim g=lim h=L ⟹ lim f=L
L'Hôpital0/0 hay ∞/∞: lim f/g = lim f'/g'

Giới hạn đặc biệt

lim_{x→0} sin(x)/x = 1
lim_{x→0} (1−cos x)/x² = 1/2
lim_{x→∞} (1+1/x)^x = e
lim_{x→0} (eˣ−1)/x = 1
lim_{x→0} ln(1+x)/x = 1
lim_{x→0} xˣ = 1  // 0⁰ dạng bất định
Liên tục (Continuity) Foundation
f liên tục tại a ⟺
(1) f(a) xác định
(2) lim_{x→a} f(x) tồn tại
(3) lim_{x→a} f(x) = f(a)

Infinite
Vô cực		Giới hạn = ±∞ (asymptote đứng)
Oscillating

Đạo hàm (Derivative) — Vi phân (Differential) Calculus I
Định nghĩa:
f'(x) = lim_{h→0} [f(x+h) − f(x)] / h

Vi phân:
df = f'(x) dx

Ký hiệu:
f'(x) = df/dx = Df = ẋ (theo t)

Quy tắc đạo hàm

Quy tắcCông thức
Sum (Tổng)(f±g)' = f'±g'
Product (Tích)(fg)' = f'g + fg'
Quotient (Thương)(f/g)' = (f'g−fg')/g²
Chain (Dây chuyền)(f∘g)' = f'(g)·g'
Inverse (Nghịch)(f⁻¹)'(y) = 1/f'(x)
Implicit (Ẩn)Đạo hàm cả hai vế theo x
Logarithmicy = uᵛ: ln y = v·ln u rồi đạo hàm

Bảng đạo hàm chuẩn

(xⁿ)' = nxⁿ⁻¹
(eˣ)' = eˣ    (aˣ)' = aˣ ln a
(ln x)' = 1/x    (log_a x)' = 1/(x ln a)
(sin x)' = cos x    (cos x)' = −sin x
(tan x)' = sec²x    (cot x)' = −csc²x
(sec x)' = sec x tan x
(arcsin x)' = 1/√(1−x²)
(arccos x)' = −1/√(1−x²)
(arctan x)' = 1/(1+x²)
(sinh x)' = cosh x    (cosh x)' = sinh x
(tanh x)' = sech²x
Định lý giá trị trung bình MVT
Rolle: f(a)=f(b) ⟹ ∃c∈(a,b): f'(c)=0

MVT (Lagrange):
∃c∈(a,b): f'(c) = [f(b)−f(a)]/(b−a)

Cauchy MVT:
∃c: f'(c)/g'(c) = [f(b)−f(a)]/[g(b)−g(a)]

Taylor's theorem:
f(x) = Σ_{k=0}^{n} f^(k)(a)/k! · (x−a)^k + R_n
R_n = f^(n+1)(c)/(n+1)! · (x−a)^(n+1)
// Lagrange remainder, c∈(a,x)

Tích phân bội & Tích phân đường/mặt Multiple Integrals

∬_D f(x,y) dA = ∫∫ f dy dx

Đổi biến — Jacobian:
∬ f(x,y)dA = ∬ f(r,θ) |J| dr dθ
J = ∂(x,y)/∂(r,θ) = r  // tọa độ cực

Triple integral:
∭ f dV — hình cầu: dV = ρ²sinφ dρdφdθ

∫_C f ds = ∫_a^b f(r(t))|r'(t)| dt
∫_C F·dr = ∫_a^b F(r(t))·r'(t) dt

∬_S F·dS = ∬_D F·(rᵤ×rᵥ) du dv
Giải tích vectơ — Các định lý lớn Vector Calculus

Toán tử vi phân vectơ

Gradient:   ∇f = (∂f/∂x, ∂f/∂y, ∂f/∂z)
Divergence: ∇·F = ∂Fₓ/∂x + ∂F_y/∂y + ∂Fz/∂z
Curl:       ∇×F = det|i j k / ∂ₓ ∂_y ∂_z / Fₓ F_y Fz|
Laplacian:  ∇²f = ∂²f/∂x²+∂²f/∂y²+∂²f/∂z²

Identities:
∇×(∇f) = 0  // curl of gradient = 0
∇·(∇×F) = 0  // div of curl = 0

ODE — Phương trình vi phân thường ODE
▸Bậc nhất (1st order)
  ├▸Tách biến: f(y)dy = g(x)dx
  ├▸Tuyến tính: y'+P(x)y = Q(x)
  ├▸Bernoulli: y'+P(x)y = Q(x)yⁿ
  └▸Exact: M dx + N dy = 0, M_y=N_x
▸Bậc hai tuyến tính (2nd order linear)
  ├▸Hệ số hằng: ay''+by'+cy = 0
  ├▸Euler–Cauchy: x²y''+xy'+y = 0
  └▸Variation of parameters (Biến thiên tham số)
▸Hệ phương trình vi phân
  └▸x' = Ax, giải theo eigenvalues
Integrating factor (Nhân tử tích phân):
